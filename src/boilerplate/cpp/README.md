# Competitive Programming Project

This project was generated by PrepKit for competitive programming.

## Structure

- `main.cpp` - Main solution file with fast I/O setup
- `includes/` - Modular utility headers
  - `utils.hpp` - Common utility functions (GCD, modular arithmetic, etc.)
  - `data_structures.hpp` - Segment tree, Fenwick tree, Union-Find
  - `graph.hpp` - Graph algorithms (DFS, BFS, Dijkstra, etc.)
- `tests/` - Sample input/output files
- `prepkit_config.yaml` - PrepKit configuration for this project

## Usage

### Compile and run
```bash
g++ -o solution main.cpp
./solution < tests/sample_input.txt
```

### Preprocess for submission (single file)
```bash
# From project root
prepkit cpp preprocess main.cpp -I ./includes
```

### Test with PrepKit
```bash
# Compile preprocessed version
g++ -o solution_processed <(prepkit cpp preprocess main.cpp -I ./includes)
./solution_processed < tests/sample_input.txt
```

## Development Tips

1. Write your solution in `main.cpp`
2. Use the utility functions from `includes/` headers
3. Test with sample input/output in `tests/`
4. Use PrepKit to generate single-file version for submission
5. The preprocessor will automatically inline all local includes

## Available Utilities

- **utils.hpp**: GCD, LCM, modular arithmetic, array utilities
- **data_structures.hpp**: Fenwick Tree, Segment Tree, Union-Find
- **graph.hpp**: DFS, BFS, Dijkstra, topological sort, bipartite check

All utilities are optimized for competitive programming and use appropriate data types.
